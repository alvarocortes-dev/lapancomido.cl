### 2. HomePage - BentoGrid Section
**File:** `apps/web/src/pages/HomePage.jsx`

**Current issue:** Tamaños inline fijos, grid-template-areas complejo

```jsx
// Problema actual:
style={{
  gridTemplateColumns: "repeat(4, 1fr)",
  gridTemplateRows: "repeat(3, 1fr)",
  width: "120rem",
  height: "55rem",
}}
```

**Solution:** Grid responsive con Tailwind

```jsx
// Mobile (1 col): Stack de imágenes
// Tablet (2 cols): Grid 2x5
// Desktop (4 cols): BentoGrid original

// Opción A: Simplificar a grid uniforme responsive
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-4">
  {images.slice(0, 8).map(...)}
</div>

// Opción B: Mantener bento pero con clases Tailwind
// (más complejo, requiere media queries custom)
```

**Subtasks:**
- [ ] Eliminar estilos inline con tamaños fijos
- [ ] Implementar grid responsive: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-4`
- [ ] Ajustar aspect-ratio de imágenes para consistencia
- [ ] Limitar max-width del container: `max-w-7xl mx-auto`
- [ ] Reducir cantidad de imágenes en móvil si es necesario
