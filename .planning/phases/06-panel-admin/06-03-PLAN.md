# Plan 06-03: Edición Contenido Home + Configuración Precios

## Goal
Admin puede configurar visibilidad de precios y editar configuración de la tienda.

## Prerequisites
- Plans 06-01 and 06-02 complete
- store_config table exists in database

## Tasks

### API: Store Config Controller
1. Create `src/controllers/storeConfig.controller.js`:
   - `getStoreConfig()` - returns current config (creates default if missing)
   - `updateStoreConfig(data)` - updates config fields

2. Add routes to `admin.routes.js`:
   - GET /admin/config
   - PUT /admin/config

### Admin Frontend: API Client
3. Create `src/api/config.js`:
   - getConfig(), updateConfig()

### Admin Frontend: Config Page
4. Expand `src/pages/SettingsPage.jsx` or create `src/pages/ConfigPage.jsx`:
   - Toggle: Mostrar precios (show_prices)
   - Input: Número WhatsApp (whatsapp_number)
   - Textarea: Mensaje de saludo (greeting)
   - Save button with loading state

### Admin Frontend: Navigation
5. Add Configuración tab if creating new page (or merge into Settings)

## Files to Create
- `apps/api/src/controllers/storeConfig.controller.js`
- `apps/admin/src/api/config.js`

## Files to Modify
- `apps/api/src/routes/admin.routes.js` - add config routes
- `apps/admin/src/pages/SettingsPage.jsx` - add config section OR
- `apps/admin/src/main.jsx` - add ConfigPage if separate

## Out of Scope (defer to later)
- Home page content editing (hero images, about text) - would require new schema
- SEO settings - handled in Phase 9
- Email templates - not needed per requirements

## Verification
- [ ] GET /admin/config returns store configuration
- [ ] PUT /admin/config updates configuration
- [ ] Toggling show_prices reflects on public catalog
- [ ] WhatsApp number change reflects in quotation links
- [ ] Greeting text change reflects in WhatsApp messages

## Estimated Duration
15-20 minutes
